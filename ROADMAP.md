# Roadmap

This file is the planning surface for Marcus's simpler job application helper.

## Next Decisions

- Define the primary workflow: tracker, CV tailoring, application drafting, interview prep, or a smaller combination.
- Decide the source of truth for personal profile, CV content, target roles, and deal-breakers.
- Choose the first interface: markdown-first, CLI scripts, local web app, or a hybrid.
- Identify which career-ops ideas are worth borrowing from `inspo/career-ops/`.

## Candidate V1 Shape

- Simple profile and CV source files.
- Lightweight application tracker.
- Focused job evaluation notes.
- Optional tailored CV or cover-letter drafts.
- No batch processing, portal scanning, dashboard, or multi-language mode system unless they become explicitly useful.

## Reference Areas

- `inspo/career-ops/modes/` for prompt and workflow ideas.
- `inspo/career-ops/templates/` for CV output inspiration.
- `inspo/career-ops/data/` and `inspo/career-ops/reports/` for tracker and report examples.
- `inspo/career-ops/providers/` and `inspo/career-ops/scan.mjs` only if scanning becomes part of the new vision.
